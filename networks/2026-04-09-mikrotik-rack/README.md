## English Version

### Project Overview

Physical reorganization of the communication rack and logical reconfiguration of the network to improve performance and security.

### Tasks Performed:

* **Cable Management:** Organized and labeled ethernet runs for better identification.
* **Bridge Configuration:** Implemented a new bridge interface to unify local traffic.
* **IP Addressing:** Reassigned subnets for better device segmentation (VLAN ready).
* **Hardware Maintenance:** Cleaned and verified SFP/Ethernet ports.

## Versión en Español

### Resumen del Proyecto

Reorganización física del rack de comunicaciones y reconfiguración lógica de la red para mejorar el rendimiento y la seguridad.

### Tareas Realizadas:

* **Gestión de Cableado:** Organización y etiquetado de cables para mejor identificación.
* **Configuración de Bridge:** Implementación de una nueva interfaz bridge para unificar el tráfico local.
* **Direccionamiento IP:** Reasignación de subredes para una mejor segmentación (listo para VLANs).
* **Mantenimiento de Hardware:** Limpieza y verificación de puertos SFP y Ethernet.
